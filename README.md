shows gathering icons
- triangles for mines
- squares for herbs
- color coded by difficulty

requires:
- pfQuest
- enter `/db mines` / `/db mines auto` or `/db herbs` / `/db herbs auto` in chat to show icons on map (`auto` selects green-orange items)
